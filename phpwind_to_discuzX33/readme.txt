本工具可以轻松转换phpwind8.x论坛到Discuz!X3.3论坛。有如下特点

1.完整转换 会员，头像，积分，主题，帖子，附件，版块，主题分类，公告，勋章，表情，友情链接，在线时间，短消息，用户组，日志，日志分类，评论，相册等相关所有重要数据。
2.出色的帖子细节处理，针对html代码，各种附件标签，多媒体标签作了特别转换。
3.智能的转换时间预估，让转换等待不再遥遥无期如同等待戈多，让你有闲暇的时间可以泡杯清茶。
4.专门针对大论坛设计的增量转换，方便超大数据量的用户分次转换，极大减少转换停站时间。


说明：
本工具为试用版，提供给中小论坛使用者用以转换论坛。
适用范围：会员≤100万  && 主题≤100万 && 帖子≤500万

超过这一适用范围的论坛需要转换，可以联系作者进行转换（有偿）。


转换步骤
1.搭建转换环境
2.建立一个空discuz!X3.3论坛。
3.运行转换程序，填入各项设置。选择需要转换的项目，推荐完整转换。
4.点开始转换，静静等待转换完成提示出现即可！
5.转换完成后，程序目录下生成ReportLog.txt，ErrorLog.txt，报告转换结果。
6.转移附件。将pw的附件目录attachment更名为forum覆盖dx论坛data\attachment\forum同名目录。
7.转移勋章。将pw论坛的images/medal/small/下所有文件夹，复制到dx论坛的static/image/common/下。
8.转移表情。将pw论坛的images/post/smile/下所有文件夹，复制到dx论坛的static\image\smiley\下。
9.转移头像。将工具同目录下生成的avatar目录覆盖dx论坛的\uc_server\data\avatar同名目录即可。
10.修改创始人。

打开dx论坛config\config_global.php文件
查找
$_config['admincp']['founder'] = '1';
改为
$_config['admincp']['founder'] = '你的uid';

11.用原pw管理员的帐号和密码登录dx论坛，依次进入“管理中心”，“工具”，“更新统计”，“重建(论坛/群组)帖数”，“重建主题帖数”，“更新缓存”；然后即可看到转换效果。
12.如选择转换日志或相册，登录后台，全局，站点功能，功能模块，开启日志，相册两项。
13.如选择转换相册，将pw论坛attachment目录下的photo文件夹复制到dx论坛的data\attachment\album下。
14.如选择转换日志，将pw论坛attachment目录下的diary文件夹复制到dx论坛根目录下。
